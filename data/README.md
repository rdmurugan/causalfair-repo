# Data

This directory is a placeholder for HMDA data files. Real HMDA data can be downloaded from the CFPB Data Browser:

https://ffiec.cfpb.gov/data-browser/

The analysis script in `examples/download_hmda_real.py` automates the download for New York State, 2022.

Note: HMDA CSV files are large (100MB+) and are excluded from version control via `.gitignore`.
